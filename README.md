An archive of all the songs and participants in the music playlist guessing game Poger.

## What is Poger?

**Poger** is an activity started by my friend group in March of 2022, when a bunch of us recommended songs that we liked and put it into a shared playlist. From there, we slowly turned it into a fun social activity that you can do as well! A typical poge consists of these steps:

- Each participant puts in 4-6 (typically 5) songs into a collaborative Spotify playlist.
- After that, someone streams the shuffled playlist to everyone.
- Enjoy a mix of everyone's music.

That's really all there is to it!
